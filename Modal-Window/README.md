# Modal Window Project

This is the second small project, focused on implementing a modal window.

## Overview

The user can open a modal by clicking on one of the buttons at the top of the page. When a modal is opened, an overlay appears, covering the rest of the page.

## Features

### Opening the Modal
- The modal is triggered by clicking any of the buttons at the top.

### Closing the Modal
The modal and overlay can be closed in multiple ways:
- Clicking the close button in the top-right corner of the modal
- Clicking outside the modal (on the overlay)
- Pressing the `Escape` key on the keyboard

## Learning Objective

This project is designed to help understand how to work with CSS classes. Class manipulation is a fundamental technique in web development, as it is one of the primary ways to dynamically control and update web page behavior and styling.
